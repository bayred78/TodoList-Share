📅 구글 캘린더 연동

이 프로젝트의 체크리스트를 구글 캘린더의 특정 캘린더와 연동합니다. 📅 버튼을 누르면 날짜를 선택할 수 있고, 선택한 날짜에 할일이 등록됩니다.

캘린더 ID 찾는 방법
  1. Google Calendar 접속
  2. 왼쪽 사이드바에서 연동할 특정 캘린더의 ⋮ 클릭
  3. "설정 및 공유" 선택
  4. "캘린더 통합" → "캘린더 ID" 복사
💡 특정 캘린더란? 구글 캘린더는 여러 개의 캘린더(업무, 개인, 공유 등)를 가질 수 있습니다. 할일을 등록할 캘린더를 선택하여 ID를 입력하세요.
📅 사용 방법: 체크리스트 항목의 📅 버튼 → 날짜 선택 → 해당 날짜에 할일이 등록됩니다.
